## Getting started

UpCloud provides computing, storage and networking needed to develop and run applications and workloads in the cloud. We also offer managed services such as [Kubernetes](/docs/products/managed-kubernetes.md), [MySQL](/docs/products/managed-mysql.md), [PostgreSQL](/docs/products/managed-postgresql.md), [Valkey](/docs/products/managed-valkey.md), and [OpenSearch](/docs/products/managed-opensearch.md). UpCloud is available in 15 locations worldwide.

[Learn more about UpCloud](/docs/getting-started/introduction.md)
