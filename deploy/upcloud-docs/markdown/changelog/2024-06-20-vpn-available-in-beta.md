# VPN Gateways available in beta

June 20, 2024
·
[Permalink](/docs/changelog/2024-06-20-vpn-available-in-beta.md)

[VPN Gateways](/docs/products/vpn-gateway.md) are now available for all customers, providing a powerful tool to enhance connectivity and network management. This new feature enables the connection of SDN Private Networks to remote networks through site-to-site IPsec connections, ensuring secure and reliable communication channels over untrusted network environments. With VPN Gateways, you can extend your private network's reach, seamlessly integrating remote sites, data centers, or cloud environments into a cohesive network infrastructure.

In addition to IPsec connectivity, VPN Gateways come equipped with Network Address Translation (NAT) features. This functionality enables Internet connectivity from the associated SDN Private Networks.

Get started by following our [How to set up site-to-site VPN connection](/docs/guides/set-up-site-to-site-vpn-connection.md) guide!
