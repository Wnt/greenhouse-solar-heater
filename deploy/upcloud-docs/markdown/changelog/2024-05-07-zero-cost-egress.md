# Introducing Zero-cost Egress

May 7, 2024
·
[Permalink](/docs/changelog/2024-05-07-zero-cost-egress.md)

Starting from today, Zero-cost egress replaces UpCloud's Network Transfer Pool. With this move, egress transfer exceeding the fair amount is no longer automatically charged, alleviating pains from unexpected costs due to unexpected amounts of transfer. Read more about [Zero-cost Egress](https://upcloud.com/products/zero-cost-egress/) and [Network transfer](/docs/products/networking/network-transfer.md).
