# Launched Managed Databases for OpenSearch

May 11, 2023
·
[Permalink](/docs/changelog/2023-05-11-managed-databases-opensearch.md)

We're thrilled to announce the launch of Managed Databases for OpenSearch, a robust open-source search and analytics suite. This new service offers scalable, highly available, and secure databases, ideal for log management, application search, or document storage. With automated backups and secure access via the private UpCloud Utility network, you can start building secure, high-performance applications today.
