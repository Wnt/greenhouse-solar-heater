# NAT Gateway released

February 20, 2023
·
[Permalink](/docs/changelog/2023-02-20-nat-gateway.md)

NAT Gateway enables outbound internet access for servers without the need for a dedicated public IP address. Whether you need to connect to third-party services or fetch software upgrades, NAT Gateway has you covered. Operating in conjunction with an SDN Router, it allows all Cloud Servers connected to an SDN Private Network to access external networks. As a managed service, NAT Gateway includes software upgrades from UpCloud and high availability in Standard, Production, and Advanced plans.
