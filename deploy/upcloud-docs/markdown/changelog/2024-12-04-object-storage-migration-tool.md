# Object Storage migration tool now available in the Control Panel

December 4, 2024
·
[Permalink](/docs/changelog/2024-12-04-object-storage-migration-tool.md)

The Object Storage migration tool is now available directly through the UpCloud Control Panel - making it easier to migrate your data.

The tool provides an interface for transferring your data from our legacy Object Storage or any S3-compatible storage provider, such as AWS S3. Users can choose to migrate either all buckets or selected ones through the Control Panel interface.

This new addition to our Control Panel works alongside our existing [API migration tool](/docs/guides/managed-object-storage-migration-tool.md), offering the same functionality such as progress monitoring and automatic bucket creation. For those planning to migrate from our legacy Object Storage service before its end-of-life in February 2025, this tool helps transition to our new Managed Object Storage.

You can access the migration tool from the UpCloud Control Panel under [Object Storage](https://hub.upcloud.com/object-storage/migrate).
