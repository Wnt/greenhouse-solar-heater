# Ubuntu 24.04 LTS template available

April 26, 2024
·
[Permalink](/docs/changelog/2024-04-26-ubuntu2204-release.md)

Ubuntu 24.04 LTS "Noble Numbat" was released yesterday, and it was added as a new public template on UpCloud today.
This new Long Term Support release includes the new Linux Kernel version 6.8, in addition to other important upgrades.
As an LTS release, standard security updates for software packages in this release are available from Ubuntu until June 2029.
Read the full [Release Notes](https://discourse.ubuntu.com/t/noble-numbat-release-notes/39890/1) from Ubuntu.
