# Custom domain support for Object Storage

August 20, 2024
·
[Permalink](/docs/changelog/2024-08-20-managed-object-storage-custom-domains.md)

UpCloud Managed Object Storage now supports customer-defined domains, enabling you to personalise your content delivery. This feature can be accessed through the Control Panel, our [API](https://developers.upcloud.com/1.3/21-managed-object-storage/#attach-custom-domain), and SDKs (coming soon).
