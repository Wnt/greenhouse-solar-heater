September 6, 2024
·
[Permalink](/docs/changelog/2024-09-11-developer-cloud-server-plans.md)

We've extended our Cloud Server offering with new Developer plans! Starting at just €3 per month, these plans bring the best value in the industry from a trusted cloud provider.

The new plan category includes options up to 16 GB of RAM, ensuring scalability for larger projects and development environments. Each plan comes with [zero-cost egress](https://upcloud.com/products/zero-cost-egress/) and a public IPv4 address.

Whether you're working on personal projects or development workloads, these plans provide the perfect balance of performance and price. And when you're ready to go into production, you can upgrade to General Purpose, High CPU and High Memory plans for more performance and our full 99.999% SLA.
