# Kubernetes v1.29 now available

September 11, 2024
·
[Permalink](/docs/changelog/2024-09-11-kubernetes-1.29.md)

We're thrilled to announce that Kubernetes 1.29 is now available on UpCloud's Managed Kubernetes service.

For a full list of available features and improvements, please see [Kubernetes 1.29 release notes](https://kubernetes.io/blog/2023/12/13/kubernetes-v1-29-release/).

UpCloud Managed Kubernetes clusters running Kubernetes v1.29 also have an upgraded version of Cilium (v1.16.1).

You can leverage Kubernetes 1.29 on UpCloud's Managed Kubernetes service by deploying a new cluster in the [UpCloud control panel](https://hub.upcloud.com/kubernetes/new).
