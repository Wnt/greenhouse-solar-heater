# Hot Resizing for Cloud Servers

June 29, 2022
·
[Permalink](/docs/changelog/2022-06-29-hot-resizing.md)

Cloud Servers can now be scaled vertically without downtime. The number of CPU cores and the amount of memory can be increased without first shutting the server down. This new feature is available on all servers created on or after April 27th, 2022. For Cloud Servers created before that, a one-time full stop and start activates the feature. Hot resizing makes scaling up effortless and provides for an uninterrupted service when more capacity is needed.
