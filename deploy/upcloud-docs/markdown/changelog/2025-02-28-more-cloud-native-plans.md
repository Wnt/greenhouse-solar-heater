# More Cloud Native plans!

February 28, 2025
·
[Permalink](/docs/changelog/2025-02-28-more-cloud-native-plans.md)

Remember those awesome [Cloud Native configurations](/docs/products/cloud-servers/plans#cloud-native-plans.md) for Cloud Servers we launched in December? Well, we heard your feedback and decided to double down! Announcing the second wave of Cloud Native plans, with even more options to fuel your cloud-native dreams. We've added some seriously sweet deals, especially in the popular 16 GB to 64 GB memory range.

Cloud Native plans work best in private clusters with Kubernetes, and they can be connected to other resources through SDN Private Networks.

Cloud Native plans come with no bundled storage or IP addresses. To be able to function, a server requires some storage, and the size and flavour can be freely picked from any of our [storage tiers](/docs/products/block-storage/tiers.md).

See [Cloud Server configurations](/docs/products/cloud-servers/configurations.md) for details.
