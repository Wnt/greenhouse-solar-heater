# SDN private network connectivity now available for Managed Databases

February 16, 2024
·
[Permalink](/docs/changelog/2024-02-16-managed-databases-sdn-support.md)

Our Managed Databases now support SDN Private Networks, marking a significant enhancement in the security and connectivity of your database services. This update applies to all our Managed Database products, including MySQL, PostgreSQL, OpenSearch, and Redis, providing an advanced layer of protection and performance optimisation.

Key enhancements:

- Private network connectivity: With the new SDN Private Network support, your Managed Databases can now benefit from fully private connections. This crucial update safeguards your data by ensuring it remains isolated from the public internet.
- Effortless cloud integration: The process of connecting your Managed Databases to your cloud infrastructure has been streamlined through an intuitive user interface, making it accessible to all users, regardless of their technical expertise.
- Terraform compatibility: For those who manage their infrastructure as code, this update also introduces Terraform support for SDN Private Networks, facilitating more efficient setup and management.
- Enhanced cloud ecosystem: This feature promotes tighter integration with other SDN-enabled services at UpCloud, creating a more secure, efficient, and interconnected cloud environment.

To discover how to set up private network connectivity on new or existing Managed Databases and unlock the full potential of this feature, refer to our step-by-step [guide](/docs/guides/connect-managed-databases-sdn-private-networks.md).
