## Upgraded Block Storage tiers available

September 3, 2024
·
[Permalink](/docs/changelog/2024-09-03-block-storage-upgrade.md)

UpCloud's block storage offering has been expanded with the introduction of the new Standard storage tier, available at an unbeatable price of only €0.085 per GB per month.

|  | MaxIOPS | Standard | Archive |
| --- | --- | --- | --- |
| Use case | High performance | General purpose | High capacity |
| Capacity | 1 GB – 4 TB | 1 GB - 4 TB | 1 GB – 4 TB |
| Performance (4k block size) | Read: 100 000 IOPS Write: 30 000 IOPS | Read: 10 000 IOPS Write: 10 000 IOPS | Read: 600 IOPS Write: 600 IOPS |
| Availability | All locations | All locations | All locations |

Standard storage is suitable for all general purpose needs, from production workloads to long term cold storage. For high-performance requirements, we recommend our proprietary MaxIOPS technology. Read more about [block storage tiers](/docs/products/block-storage/tiers.md) on our website.

All three block storage tiers, including the new Standard tier, are available at all UpCloud locations.

Additionally, we've upgraded the hardware for our HDD tier which we will be now renaming to *Archive*. New devices will be created on a more performant storage system, offering improved stability.

As before, devices from multiple storage tiers can be used on the same server, and each server supports a maximum of 16 storage devices. All block storage devices can be moved freely between servers within the same location.

## Encryption and flexible plan support for UKS

August 13, 2024
·
[Permalink](/docs/changelog/2024-08-13-k8s-encryption-and-flexible-plans.md)

UpCloud Managed Kubernetes now supports encryption-at-rest with various configurations. Users can
define encryption on cluster level, node group level and per Persistent Volume. Learn more from the [encrypted clusters documentation](/docs/products/managed-kubernetes/encrypted-clusters.md).

We have also added support for flexible plans in cluster node groups. Read more from the Terraform `upcloud_kubernetes_node_group` [resource documentation](https://registry.terraform.io/providers/UpCloudLtd/upcloud/latest/docs/resources/kubernetes_node_group#nested-schema-for-custom_plan).
