## Launch of UpCloud Managed Kubernetes

April 26, 2023
·
[Permalink](/docs/changelog/2023-04-26-managed-kubernetes.md)

Following a successful Open Beta, we're delighted to officially launch UpCloud Managed Kubernetes (UKS). This service simplifies the deployment of containerised applications by managing the underlying infrastructure. UKS takes the complexity out of the control plane, freeing DevOps teams to focus on optimising their deployment pipelines. Experience the full power of containerisation with the official release of UpCloud Managed Kubernetes!

## NAT Gateway released

February 20, 2023
·
[Permalink](/docs/changelog/2023-02-20-nat-gateway.md)

NAT Gateway enables outbound internet access for servers without the need for a dedicated public IP address. Whether you need to connect to third-party services or fetch software upgrades, NAT Gateway has you covered. Operating in conjunction with an SDN Router, it allows all Cloud Servers connected to an SDN Private Network to access external networks. As a managed service, NAT Gateway includes software upgrades from UpCloud and high availability in Standard, Production, and Advanced plans.

Read more about [NAT Gateways](/docs/products/nat-gateway.md)

## Transition to Euro (EUR) as Base Currency

January 1, 2023
·
[Permalink](/docs/changelog/2023-01-01-changed-base-currency-to-eur.md)

Starting January 1st, 2023, we are transitioning our pricing currency from US Dollars (USD) to Euros (EUR) to align more closely with our customer base and streamline financial processes. This change aims to mitigate currency fluctuation uncertainties, enhance cost planning and price predictability, and simplify invoicing and payments.

Read more about the [base currency change](https://upcloud.com/blog/new-cloud-server-plans-improvements-pricing-adjustments/)
