## UpCloud Managed Kubernetes now available in Private Cloud

May 20, 2024
·
[Permalink](/docs/changelog/2024-05-20-uks-private-cloud.md)

Starting today, UpCloud Managed Kubernetes is available on all [Private Cloud](/docs/products/private-cloud.md) zones! Kubernetes data plane, or worker nodes, can be deployed into your fleet of Private Cloud hosts, providing enhanced security and control. Read more about [Managed Kubernetes in Private Cloud](/docs/products/managed-kubernetes/private-cloud.md).

## UpCloud Crossplane Provider released

May 14, 2024
·
[Permalink](/docs/changelog/2024-05-14-crossplane-release.md)

With the initial release of UpCloud Crossplane Provider, users can now manage UpCloud resources (such as servers, storage, networks, etc.) directly through [Crossplane](https://www.crossplane.io), using Kubernetes manifests, even across the cloud! This complements our Infrastructure-as-Code (IaC) tooling by providing a Kubernetes-native way to manage UpCloud resources. Read more about our [Crossplane Provider](/docs/tooling/crossplane-with-upcloud.md).

## Introducing Zero-cost Egress

May 7, 2024
·
[Permalink](/docs/changelog/2024-05-07-zero-cost-egress.md)

Starting from today, Zero-cost egress replaces UpCloud's Network Transfer Pool. With this move, egress transfer exceeding the fair amount is no longer automatically charged, alleviating pains from unexpected costs due to unexpected amounts of transfer. Read more about [Zero-cost Egress](https://upcloud.com/products/zero-cost-egress/) and [Network transfer](/docs/products/networking/network-transfer.md).

## Encryption at Rest released

April 16, 2024
·
[Permalink](/docs/changelog/2024-04-16-encryption-at-rest-release.md)

[Encryption at Rest](/docs/products/block-storage/encryption-at-rest.md) for Cloud Servers is now available. Encryption is supported both in MaxIOPS and HDD block storage tiers.

Encryption can be activated on any storage device upon creation, ensuring that all data written to the storage device is encrypted using AES-256. The management of encryption keys is handled entirely by UpCloud. This feature is provided at no extra cost.

![Encryption at Rest is enabled when creating a storage](../../../products/block-storage/encryption-at-rest/storage-creation-encryption-at-rest.png)

With this enhancement, all data stored on UpCloud can benefit from encryption. Encryption at Rest is supported across block storages, Managed Databases, and Object Storage services.
