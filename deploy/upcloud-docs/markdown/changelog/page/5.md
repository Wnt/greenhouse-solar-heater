## More Cloud Native plans!

February 28, 2025
·
[Permalink](/docs/changelog/2025-02-28-more-cloud-native-plans.md)

Remember those awesome [Cloud Native configurations](/docs/products/cloud-servers/plans#cloud-native-plans.md) for Cloud Servers we launched in December? Well, we heard your feedback and decided to double down! Announcing the second wave of Cloud Native plans, with even more options to fuel your cloud-native dreams. We've added some seriously sweet deals, especially in the popular 16 GB to 64 GB memory range.

Cloud Native plans work best in private clusters with Kubernetes, and they can be connected to other resources through SDN Private Networks.

Cloud Native plans come with no bundled storage or IP addresses. To be able to function, a server requires some storage, and the size and flavour can be freely picked from any of our [storage tiers](/docs/products/block-storage/tiers.md).

## Cloud Native plans for Cloud Servers

December 12, 2024
·
[Permalink](/docs/changelog/2024-12-12-cloud-native-plans.md)

UpCloud introduces a new line of Cloud Native plans for Cloud Servers, designed to deliver exceptional price-performance for modular workloads, containerized applications, and microservices architectures. These plans offer a streamlined approach to cloud server provisioning, focusing on core compute resources (CPU and memory) and allowing you to tailor storage and networking independently to match your specific needs.

**Unbundled Storage:** Choose the storage tier and size that best suits your workload, with no fixed storage included in the base plan. This provides greater flexibility and cost optimization.

**Hourly Billing:** Pay only for the compute resources you consume, with no long-term commitments. Ideal for dynamic and short-lived workloads. Cloud Native plans are only billed when the server is powered on.

**Optimized for Modern Architectures:** Perfect for containerized applications, microservices, and cloud-native environments where servers are treated as ephemeral resources.

**Unbeatable Price-Performance:** Our Cloud Native plans offer the best possible pricing against hypervendors like AWS and GCP, allowing you to maximize your cloud budget without compromising on performance.

Read more about different [Cloud Server plans and configurations](/docs/products/cloud-servers/configurations.md).

## Object Storage migration tool now available in the Control Panel

December 4, 2024
·
[Permalink](/docs/changelog/2024-12-04-object-storage-migration-tool.md)

The Object Storage migration tool is now available directly through the UpCloud Control Panel - making it easier to migrate your data.

The tool provides an interface for transferring your data from our legacy Object Storage or any S3-compatible storage provider, such as AWS S3. Users can choose to migrate either all buckets or selected ones through the Control Panel interface.

This new addition to our Control Panel works alongside our existing [API migration tool](/docs/guides/managed-object-storage-migration-tool.md), offering the same functionality such as progress monitoring and automatic bucket creation. For those planning to migrate from our legacy Object Storage service before its end-of-life in February 2025, this tool helps transition to our new Managed Object Storage.

You can access the migration tool from the UpCloud Control Panel under [Object Storage](https://hub.upcloud.com/object-storage/migrate).

## Managed Object Storage now available in all regions

November 21, 2024
·
[Permalink](/docs/changelog/2024-11-21-managed-object-storage-launched-usa.md)

Starting today, UpCloud Managed Object Storage is available in Chicago (US-CHI1), completing our US region coverage and our global rollout of the service. This expansion joins our existing availability in Europe and Asia, providing enhanced storage capabilities across all major regions.

The Managed Object Storage service offers:

- Full S3 compatibility for seamless integration with existing tools
- Built-in AES-256 encryption for data at rest and in transit
- Automatic scaling starting from 250GB
- Custom domain support
- Direct connectivity through SDN Private Networks

For customers currently using our legacy Object Storage service, we recommend migrating to the new Managed Object Storage before February 28th, 2025, when the legacy service will reach its end-of-life. To assist with the transition, we provide detailed migration guides:

- [How to migrate from our legacy Object Storage](/docs/guides/managed-object-storage-migration-tool.md)
- [How to migrate from AWS S3](/docs/guides/migrate-aws-s3-to-upcloud.md)

Get started with Managed Object Storage today through the [UpCloud Control Panel](https://hub.upcloud.com/object-storage/2.0) or [API](https://developers.upcloud.com/1.3/21-managed-object-storage/).
