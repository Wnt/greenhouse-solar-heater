# UpCloud Crossplane Provider released

May 14, 2024
·
[Permalink](/docs/changelog/2024-05-14-crossplane-release.md)

With the initial release of UpCloud Crossplane Provider, users can now manage UpCloud resources (such as servers, storage, networks, etc.) directly through [Crossplane](https://www.crossplane.io), using Kubernetes manifests, even across the cloud! This complements our Infrastructure-as-Code (IaC) tooling by providing a Kubernetes-native way to manage UpCloud resources. Read more about our [Crossplane Provider](/docs/tooling/crossplane-with-upcloud.md).
