# Server groups and anti-affinity released

June 7, 2023
·
[Permalink](/docs/changelog/2023-06-07-server-groups-anti-affinity.md)

With the introduction of Server Groups, you can now organise servers into groups. By enabling the anti-affinity feature on these groups, servers will be deployed on separate hardware, enhancing redundancy and reliability. This functionality ensures that critical services remain operational even in the event of hardware failures. Take advantage of server groups and anti-affinity to increase the resilience of your infrastructure and maintain uninterrupted service availability.
