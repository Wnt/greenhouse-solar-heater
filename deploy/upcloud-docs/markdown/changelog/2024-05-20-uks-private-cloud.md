# UpCloud Managed Kubernetes now available in Private Cloud

May 20, 2024
·
[Permalink](/docs/changelog/2024-05-20-uks-private-cloud.md)

Starting today, UpCloud Managed Kubernetes is available on all [Private Cloud](/docs/products/private-cloud.md) zones! Kubernetes data plane, or worker nodes, can be deployed into your fleet of Private Cloud hosts, providing enhanced security and control. Read more about [Managed Kubernetes in Private Cloud](/docs/products/managed-kubernetes/private-cloud.md).
