# UpCloud CLI

[The UpCloud CLI (upctl) - [https://github.com/UpCloudLtd/upcloud-cli) provides a command-line interface to UpCloud services. ## Installing We release pre-built packages for Linux, macOS and Windows. See [upctl installation instructions](https://upcloudltd.github.io/upcloud-cli/).

## Available commands

See [upctl command reference](https://upcloudltd.github.io/upcloud-cli/commands_reference/upctl/).
