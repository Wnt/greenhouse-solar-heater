# Crossplane with UpCloud

The UpCloud Crossplane provider extends Crossplane's control plane capabilities to provision and manage UpCloud infrastructure and cloud services directly from within Kubernetes. The provider is open source and [available in GitHub](https://github.com/UpCloudLtd/provider-upcloud).
