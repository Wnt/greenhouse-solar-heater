# Terraform with UpCloud

Terraform®, a popular Infrastructure-as-Code (IaC) tool from HashiCorp Inc., simplifies cloud infrastructure management by letting you define your resources as code. This approach ensures predictable deployments, reduces errors, and streamlines collaboration.

[The UpCloud Terraform provider](https://github.com/UpCloudLtd/terraform-provider-upcloud) extends this power to UpCloud, allowing you to control all aspects of your UpCloud environment through the convenience of code. The provider is open source.

UpCloud is a member of the HashiCorp Technology Partner Program.

## Getting started

If you are new to Terraform, please see our [Terraform guides](/docs/guides/tooling#terraform.md).

See [the provider documentation](https://registry.terraform.io/providers/UpCloudLtd/upcloud/latest/docs) for a full reference.
