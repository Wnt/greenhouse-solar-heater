# Packer with UpCloud

[The Packer® plugin for UpCloud](https://github.com/UpCloudLtd/packer-plugin-upcloud) seamlessly integrates the popular image-building tool, Packer, with UpCloud's cloud platform. This plugin lets you easily create and import custom server images, complete with your desired software and configurations, ready for rapid deployment on UpCloud.

## Getting started

See [guides](/docs/guides/tooling#packer.md) to get started.

Full plugin documentation is available on [Packer integration website](https://developer.hashicorp.com/packer/integrations/UpCloudLtd/upcloud).
