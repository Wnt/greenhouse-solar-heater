# Ansible with UpCloud

The UpCloud Ansible Collection provides an inventory to automate the management and configuration of UpCloud resources within your Ansible playbooks and roles.

Current scope only covers UpCloud's cloud server offering. Currently it does not support other resource types, such as networks, routers or databases.

See [guides](/docs/guides/tooling#ansible.md) to get started.
