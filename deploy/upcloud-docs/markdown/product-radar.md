# Product radar

The Product radar demonstrates features that we're working on currently, and soon going to be released.

Also see our [Changelog](/docs/changelog.md) for recently released features!

## Ready to use

Projects that have been recently released, in public testing or limited availability.

#### File Storage (NFS)

A distributed file system that allows users to access and share files over a network as if they were stored locally. Can be used with Cloud Servers and as persistent volumes in Kubernetes. 
[Find out more](/docs/products/file-storage.md)

#### GPU Servers

Cloud Servers with NVIDIA L40S GPUs. 
[Find out more](/docs/products/gpu-servers.md)

#### Kubernetes Lifecycle Management

Support for Kubernetes cluster version upgrades.

#### Managed Databases Dynamic Disk Sizing

More flexibility to scale your service's disk storage, without upgrading your plan.

#### Pulumi Provider

Official UpCloud support for [Pulumi](https://www.pulumi.com/), an Infrastructure-as-Code SDK.

#### WHMCS integration

Integration to manage UpCloud resources directly from [WHMCS](https://www.whmcs.com/).

#### API Authentication Tokens

Token-based authentication to use UpCloud with our [tooling](/docs/tooling.md) and 3rd party integrations.

## In progress

Projects we're actively working on now.

#### Blesta integration

Integration to manage UpCloud resources directly from Blesta, a webhosting automation platform.

#### MaxIOPS v2

Next generation of our block storage platform, with further increased performance and functionality.

#### SDN Firewall

Firewalling between SDN Private Networks and other connected networks.

#### Workspaces

Feature to centrally manage access control and grouping of multiple UpCloud user accounts and resources.

#### Zone-to-Zone SDN Private Networks

Connect SDN Private Networks between UpCloud locations.

## Planning

Projects we're planning to start implementing soon.

#### Anycast Load Balancer

A global load balancer to direct traffic to the nearest healthy server worldwide, providing faster response times and seamless application availability.

#### Container Registry

A secure and scalable container registry for storing, managing, and deploying container images across your environments at UpCloud.

#### Custom image improvements

Faster deployments and ability to share custom images between physical locations.

#### Horizontal autoscaling

Automatically adjust a group of servers based on resource utilisation.

#### Kubernetes Built-in Autoscaling

Optimize resource utilization and costs with the Kubernetes Cluster Autoscaler as a built-in, readily available feature in UKS.

#### Offsite backups

Securely store your backups in a separate physical location, protecting them from fires, floods and power outages.
