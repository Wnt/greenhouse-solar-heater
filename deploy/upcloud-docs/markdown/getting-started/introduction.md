## What is IaaS

Infrastructure-as-a-Service, or IaaS for short, is an online on-demand service that offers the fundamental components of cloud servers with computing, storage, and networking resources using a pay-as-you-go model. On IaaS, users can [deploy and scale cloud resources](/docs/guides/deploy-server.md) according to their needs without large up-front investments. For example, cloud servers can be created quickly whenever required and easily deleted when no longer needed.

IaaS also allows users to react quickly to their changing infrastructure requirements. It provides users with full control of their cloud resources and operates at the lowest level underneath other computing service models such as Platform-as-a-Service or Software-as-a-Service.

## Utilising our services

UpCloud users have the option to deploy and manage cloud servers via the easy-to-use UpCloud Control Panel, the fully-featured UpCloud API, as well as the cloud automation tool, [Terraform® provider for UpCloud](/docs/guides/upcloud-terraform-provider.md) and the [UpCloud Command-Line Interface](/docs/guides/upcloud-command-line-interface.md). All features and services are equally available both at the control panel and via the API. We also offer a number of [API clients](https://github.com/UpCloudLtd) for many popular programming languages.

Once deployed, users have full access to the cloud server’s operating system with the choices of SSH for Linux®, Remote Desktop for Windows®, VNC, and web console for [connecting to the cloud server](/docs/guides/connecting-to-your-server.md).
