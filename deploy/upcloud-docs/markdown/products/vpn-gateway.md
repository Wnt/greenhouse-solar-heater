# VPN Gateway

VPN gateways connect [SDN Private Networks](/docs/products/networking/sdn-private-networks.md) and external networks using IPsec site-to-site VPN (Virtual Private Network) connections.

## VPN Connections

Secure, encrypted tunnels can be established between the private network and other external networks. This allows for safe communication and data transfer, crucial for scenarios like connecting to a corporate network or creating a multi-site network setup. The VPN feature supports site-to-site IPsec connections.

## NAT included

All VPN Gateways can simultaneously act as a [NAT Gateway](/docs/products/nat-gateway.md).
