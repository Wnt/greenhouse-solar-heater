# Customisable Managed Valkey properties

The following Valkey™\* database properties can be set by the user via the [UpCloud API](https://developers.upcloud.com/1.3/).

- `automatic_utility_network_ip_filter`
- `ip_filter`
- `migration`
- `public_access`
- `valkey_acl_channels_default`
- `valkey_io_threads`
- `valkey_lfu_decay_time`
- `valkey_lfu_log_factor`
- `valkey_maxmemory_policy`
- `valkey_notify_keyspace_events`
- `valkey_number_of_databases`
- `valkey_persistence`
- `valkey_pubsub_client_output_buffer_limit`
- `valkey_ssl`
- `valkey_timeout`
- `valkey_version`
- `service_log`
