# Network Peering

Network peering is a powerful feature that allows you to connect multiple Virtual Private Clouds (VPCs) and create a unified network environment. With UpCloud, you can enjoy the following benefits of VPC peering:

- Combine multiple UpCloud VPCs into a larger private network for expanded reach within the same region.
- Establish secure, private links between your and any other UpCloud account resources or between compatible external cloud platforms like Aiven.io.
- Enhance security by ensuring that traffic between peered VPCs never touches the public internet.
- Experience low-latency, high-bandwidth connections, which are ideal for data transfers and collaboration between different clouds.

## Available regions

Network peering can be established between SDN private networks in different region locations:

| Peering region | Accessible SDN locations |
| --- | --- |
| Europe | FI-HEL1 FI-HEL2 DE-FRA1 DK-CPH1 ES-MAD1 NL-AMS1 NO-SVG1 PL-WAW1 SE-STO1 UK-LON1 |
| United States | US-CHI1 US-NYC1 US-SJO1 |
| Singapore | SG-SIN1 |
| Australia | AU-SYD1 |

For step-by-step instructions on creating a peering, see the [How to configure network peering](/docs/guides/configure-network-peering.md) guide.
