# File Storage

File Storage provides managed Network File System (NFS) storage that enables multiple servers to access shared files over [SDN Private Networks](/docs/products/networking/sdn-private-networks.md). Perfect for applications that require shared data access, collaborative workloads, or centralized file management across your infrastructure.
