# Managed OpenSearch Backups

Each Managed Database plan includes automated backups. The backup retention is set to either 1 day for single node plans and 3 or 7 days for all the multi-node plans.

OpenSearch® backups allow for recovery to a previous point in time through two backup frequencies: hourly and daily. Customisation of these backup settings is not currently available.

All backups are encrypted and stored off-site independently of the user’s Managed Database without using the cluster’s storage capacity.
