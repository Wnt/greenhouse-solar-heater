# Managed Databases for Valkey

Managed Databases are a carefree database-as-a-service for running well-known open-source databases in the cloud. Built on high availability, Managed Databases provide a safe way to store application data.

[Valkey™](https://valkey.io/)\* is an open-source fork of Redis, offering an in-memory data store that serves as a database, cache, streaming engine, or message broker. It maintains Redis's signature sub-millisecond response times, enabling millions of requests per second for real-time applications
