# Managed Databases for PostgreSQL

Managed Databases are a carefree database-as-a-service for running well-known open-source databases in the cloud. Built on high availability, Managed Databases provide a safe way to store application data.

[PostgreSQL](https://www.postgresql.org/)®, also known as Postgres, is a free and open-source relational database management system emphasizing extensibility and SQL compliance.
