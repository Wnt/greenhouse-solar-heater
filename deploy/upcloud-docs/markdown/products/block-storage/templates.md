# Public templates

We offer users ready-to-go templates of the most popular Linux® distributions and the latest Windows Server versions. Using the templates is the fastest way to deploy new Cloud Server instances.

## Available operating systems

Users have the choice of the following system templates:

- AlmaLinux
- CentOS Stream
- Debian
- Rocky Linux
- Ubuntu
- Windows Server Standard/Datacenter

Additionally for [GPU Servers](/docs/products/gpu-servers.md), the following templates are available:

- Ubuntu with NVIDIA® Drivers and CUDA®, includes the following pre-installed software:
  - Ubuntu Server operating system
  - NVIDIA GPU drivers
  - NVIDIA CUDA drivers
  - NVIDIA CUDA Toolkit
  - NVIDIA Container Toolkit
  - Docker® Community Edition

We strive to keep the templates up-to-date by periodically updating the templates with patches and the latest security updates. Users are advised to always update their operating system after deployment and to make sure the operating system is always kept up-to-date.

## Available installation media

Additionally, we have made the templated operating systems as well as a wide range of alternative distribution available in the form of CD-ROM installation media. These can be used to manually install any of the following operating systems:

- Arch Linux
- CentOS
- CloudLinux
- Debian
- Fedora
- FreeBSD
- Gentoo Linux
- OpenBSD
- Ubuntu
- Windows Server Standard/Datacenter

The installation media can also be used to recover operating system issues by [booting the server with a live CD](/docs/guides/booting-server-live-cd.md). Users can also import their own installation media by uploading an operating system installation image using the [Storage Import feature](/docs/guides/storage-import.md).
