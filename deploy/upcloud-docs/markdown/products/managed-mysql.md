# Managed Databases for MySQL

Managed Databases are a carefree database-as-a-service for running well-known open-source databases in the cloud. Built on high availability, Managed Databases provide a safe way to store application data.

MySQL® is the world’s most popular enterprise-grade open-source relational database management system. Managed Databases utilise [MySQL Community Edition](https://www.mysql.com/products/community/), supported by a huge and active community of open-source developers.
