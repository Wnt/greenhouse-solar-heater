# File Storage encryption

File Storage instances can be initialized with encryption support. File Storage encryption is fully transparent to any connecting server and requires no additional software installation or encryption key management.

Encryption is performed using the Advanced Encryption Standard (AES) with 256-bit keys. Encryption at rest is offered free of charge.

Encryption at rest must be enabled when creating the File Storage instance. It cannot be enabled or disabled for an existing instance. To change this setting, we recommend creating another instance and copying the files over.
