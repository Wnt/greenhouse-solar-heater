# Managed Object Storage

Our new and improved Managed Object Storage allows users to create data containers called buckets to store files as data objects.

Users can deploy and manage Object Storage instances at their [UpCloud Control Panel](https://hub.upcloud.com/) and API. It’s possible to create as many Object Storage instances as users may need for their storage capacity requirements.

Object Storage instances can be accessed anonymously via HTTPS, the UpCloud control panel and the S3-compliant API, accessed from a Cloud Server over the secure Utility Network.
