# Managed Databases for OpenSearch

Managed Databases are a carefree database-as-a-service for running well-known open-source databases in the cloud. Built on high availability, Managed Databases provide a safe way to store application data.

[OpenSearch](https://opensearch.org/) is an open-source distributed search and analytics suite. Powered by the Apache Lucene search library, OpenSearch offers a vendor-agnostic toolset you can use to build secure, high-performance, cost-efficient applications, from real-time application monitoring to website search functionality.
