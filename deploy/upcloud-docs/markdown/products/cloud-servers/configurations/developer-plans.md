# Developer plans

Developer plans have been discontinued

Developer plans have been discontinued and no longer sold to new customers. We advice to use **Starter** plans instead. Developer plan deployments through the API remain possible.

Developer server plans offer a cost-effective solution with reliable performance, perfect for experimenting and building your projects.

**Plans include**

- CPU & Memory
- Standard block storage
- 1 public IPv4 & IPv6 address
- 99.99% SLA

**Add-ons**

- Extra block storage ([any tier](/docs/products/block-storage/tiers.md))
- Extra IP addresses
- Backup options
