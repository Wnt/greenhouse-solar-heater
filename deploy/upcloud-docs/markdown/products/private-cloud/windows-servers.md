# Windows Server licensing for Private Clouds

Host-level licensing for UpCloud Private Cloud provides a predictable, high-density solution for running Windows workloads. By licensing the physical hardware rather than individual Cloud Servers, you can scale your infrastructure without a linear increase in software costs.

## How It Works

Traditionally, Windows licensing is often calculated per server or per vCPU core allocated to that server. On a Private Cloud node, users can choose to opt for a single, fixed monthly fee to the entire dedicated host.

- Unlimited instances: Once the host is licensed, user can deploy as many Windows Server VMs as they wish.
- No per-server tracking: Users do not need to calculate licenses for every new instance launched on that specific host.
- Cost efficiency: This model is specifically designed for Windows-heavy environments where the density of Cloud Servers would make individual licensing prohibitively expensive.
