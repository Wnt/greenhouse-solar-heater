# Managed PostgreSQL configurations

Managed Databases for PostgreSQL® are offered with 2 and 3-node clustered solutions as well as single-node unclustered options. The nodes can be configured with the required resources by selecting from the available plans. Each plan includes a set number of CPU cores, system memory, storage and backups.

The Managed Database plans are priced according to the number of nodes and the resources allocated to each node. Please refer to our [pricing page](https://upcloud.com/pricing/#managed-databases) for hourly and monthly costs.

Database clusters are billed hourly according to the selected configuration plan until the instance is deleted. A cluster is billed for the duration the instance exists, including while stopped.
