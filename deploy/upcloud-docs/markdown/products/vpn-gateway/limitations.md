# VPN Gateway limitations

## Site-to-site only

Only site-to-site connections are supported. The VPN service does not support end users connecting their endpoints directly to the VPN service.

## IPv4 only

Both SDN Private Networks and the VPN service only support IPv4 subnets. Using IPv6 is currently not possible.

## Internet Key Exchange versions

Only IKEv2 (Internet Key Exchange) is supported. For enhanced security, IKEv1 is not supported.
