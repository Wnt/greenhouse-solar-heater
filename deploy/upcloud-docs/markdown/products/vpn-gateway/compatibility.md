# Compatible VPN endpoints

VPN Gateway has been tested to be compatible against the following cloud providers, VPN products, hardware and services.

## Amazon Web Services

UpCloud's VPN Gateway has been tested and is compatible with the [AWS® Site-to-Site VPN](https://docs.aws.amazon.com/vpn/latest/s2svpn/VPC_VPN.md) service.

→ [Connecting a VPN Gateway to AWS Site-to-Site VPN service](/docs/guides/connect-upcloud-aws-using-vpn-gateway.md)

## Fortinet

UpCloud's VPN Gateway has been tested and is compatible with the [Fortinet FortiGate® firewall](https://docs.fortinet.com/product/fortigate/).

## MikroTik

UpCloud's VPN Gateway has been tested and is compatible with the [MikroTik RouterOS® VPN feature](https://help.mikrotik.com/docs/display/ROS/IPsec).

## Other

VPN Gateway uses the standard IPsec suite of protocols to establish VPN tunnels, making it compatible with most VPN solutions.
